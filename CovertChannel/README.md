# Covert Channel from the ["Cross-Core Covert Channel for RISC-V: Implementation, Countermeasures and Cross-Platform Analysis"](https://telecom-paris.hal.science/hal-05510026/) paper using thread counter

Cover Channel using thread counter as timerfree method to replace the `rdcycle` methods used in the original paper.

# Run Code
Run `make` then `./run`.
